Automation Project

This project was developed on myTinyToDo platform. myTinyToDo platform is a user engagement platform that enables you to create tasks, manage them in different lists, add note for each and priority, edit their properties, mark them as a Completed tasks or delete them if you wish & more.

The Goal:

Create test cases in order to test myTinyToDo platform's features and functionality.
Build solid automation using POM (Page Object Model), and Java, Selenium, TestNG and Maven tools.

Allure:

To get the best report for the Automation project and to get report readable and clear for anyone I inserted to the project Allure's features and annotations,
And now everyone can anderstand the results of the Automation run.

Code Management

To manage the code in the best way I use Git + GitHub and Jenkins, Which gives a possibility to keep and change the code in the safest way, Share it with all the team and run it automatically at regular times.
